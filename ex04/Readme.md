# 第4回
## 逃げろこうかとん(ex04/dodge_bomb.py)
### ゲーム概要
- ファイルを実行すると、1400x700のスクリーンが表示され、そこに草原の背景とこうかとん、爆弾が表示される。
- 表示された爆弾に当たらないようにこうかとんを操作するゲーム

### 追加機能
- 爆弾に画像を追加
- 爆弾がこうかとんに当たった際にGAME OVERの表示をする
